L'ecosistema Nostr sta crescendo rapidamente. Ogni giorno migliaia di nuovi utenti si uniscono alla rete, creando note, condividendo contenuti e costruendo connessioni. Ma con questa crescita arriva una sfida ben nota: **come si separa il segnale dal rumore?**

## Il Problema delle Reti Aperte

Le reti aperte e senza permessi sono potenti proprio perché chiunque può partecipare. Ma questa apertura significa anche dover affrontare:

- **Spam e account bot** che inondano il tuo feed
- **Impostori** che fingono di essere qualcun altro
- **Contenuti di bassa qualità** che sommergono le discussioni di valore

Le soluzioni tradizionali si basano sulla moderazione centralizzata — un team che decide cosa è accettabile. Ma questo vanifica lo scopo di una rete decentralizzata.

## Ecco il Web of Trust

Il Web of Trust adotta un approccio diverso. Invece di affidarsi a un'autorità centrale, sfrutta qualcosa che già possiedi: **le tue connessioni sociali**.

Il concetto è semplice:

1. **Ti fidi delle persone che segui**
2. **Loro si fidano delle persone che seguono**
3. **La fiducia fluisce attraverso queste connessioni**

Se qualcuno è a 1 hop di distanza (lo segui direttamente), è altamente fidato. A 2 hop (un amico di un amico), abbastanza fidato. 3 o più hop? Probabilmente rumore.

```javascript
// Verifica se qualcuno è nel tuo Web of Trust
const inNetwork = await window.nostr.wot.isInMyWoT(pubkey);

// Ottieni la distanza esatta
const hops = await window.nostr.wot.getDistance(pubkey);

// Ottieni un punteggio di fiducia (0-1)
const score = await window.nostr.wot.getTrustScore(pubkey);
```

## Come Funziona

La nostra estensione per browser costruisce un grafo locale delle tue connessioni sociali. Quando visiti qualsiasi app Nostr, essa può interrogare questo grafo per determinare i livelli di fiducia per qualsiasi chiave pubblica.

<Callout type="tip">
L'estensione funziona con qualsiasi app Nostr che implementi l'API WoT. Non servono modifiche al tuo client preferito!
</Callout>

### Privacy al Primo Posto

Abbiamo progettato il sistema con la privacy come principio fondamentale:

- **Modalità Locale**: Tutto resta nel tuo browser. Zero richieste esterne.
- **Modalità Remota**: Solo ricerche di chiavi pubbliche — nessun contenuto, nessun tracciamento.
- **Modalità Ibrida**: Il meglio di entrambi i mondi con la cache.
