O ecossistema Nostr está crescendo rapidamente. Todos os dias, milhares de novos usuários entram na rede, criando notas, compartilhando conteúdo e construindo conexões. Mas com esse crescimento vem um desafio familiar: **como você separa o sinal do ruído?**

## O Problema com Redes Abertas

Redes abertas e sem permissão são poderosas precisamente porque qualquer um pode participar. Mas essa abertura também significa lidar com:

- **Spam e contas bot** inundando seu feed
- **Impostores** fingindo ser alguém que não são
- **Conteúdo de baixa qualidade** afogando discussões valiosas

Soluções tradicionais dependem de moderação centralizada—uma equipe decidindo o que é aceitável. Mas isso contradiz o propósito de uma rede descentralizada.

## Entra o Web of Trust

Web of Trust adota uma abordagem diferente. Em vez de depender de uma autoridade central, ele aproveita algo que você já tem: **suas conexões sociais**.

O conceito é simples:

1. **Você confia nas pessoas que segue**
2. **Elas confiam nas pessoas que seguem**
3. **A confiança flui através dessas conexões**

Se alguém está a 1 salto de distância (você os segue diretamente), são altamente confiáveis. 2 saltos (amigo de um amigo)? Algo confiáveis. 3 ou mais saltos? Provavelmente ruído.

```javascript
// Verifique se alguém está no seu Web of Trust
const naRede = await window.nostr.wot.isInMyWoT(pubkey);

// Obtenha a distância exata
const saltos = await window.nostr.wot.getDistance(pubkey);

// Obtenha uma pontuação de confiança (0-1)
const pontuacao = await window.nostr.wot.getTrustScore(pubkey);
```

## Como Funciona

Nossa extensão de navegador constrói um grafo local das suas conexões sociais. Quando você visita qualquer app Nostr, ela pode consultar esse grafo para determinar níveis de confiança para qualquer pubkey.

<Callout type="tip">
A extensão funciona com qualquer app Nostr que implemente a API WoT. Não são necessárias mudanças no seu cliente favorito!
</Callout>

### Privacidade em Primeiro Lugar

Projetamos o sistema com privacidade como princípio fundamental:

- **Modo Local**: Tudo permanece no seu navegador. Zero requisições externas.
- **Modo Remoto**: Apenas consultas de pubkey—sem conteúdo, sem rastreamento.
- **Modo Híbrido**: O melhor dos dois mundos com cache.

## O Que Vem a Seguir

Isso é apenas o começo. Estamos trabalhando em:

- **SDK para desenvolvedores** para integrar WoT em seus apps
- **Servidor Oracle** para apps que não podem executar cálculos do lado do cliente
- **Pontuação avançada** com seguidores mútuos e bônus de ponte
