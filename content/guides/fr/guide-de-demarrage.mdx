Ce guide vous accompagne dans l'installation de l'extension Nostr WoT, la configuration de votre identite et la mise en route de votre reseau de confiance. L'ensemble du processus prend environ deux minutes.

## Installation de l'extension

L'extension est compatible avec tous les principaux navigateurs bases sur Chromium et Firefox. Choisissez votre navigateur et suivez les etapes ci-dessous.

### Chrome, Brave, Edge et Opera

1. Visitez la [page de telechargement](/download) et cliquez sur le lien du Chrome Web Store.
2. Cliquez sur **Ajouter a Chrome** (ou le bouton equivalent de votre navigateur).
3. Confirmez l'installation lorsque vous y etes invite.
4. Epinglez l'extension a votre barre d'outils pour un acces facile — cliquez sur l'icone en forme de puzzle dans votre navigateur et activez l'epingle a cote de Nostr WoT.

{/* TODO: Screenshot of the Chrome Web Store listing with the Add to Chrome button highlighted */}

### Firefox

1. Visitez la [page de telechargement](/download) et cliquez sur le lien des modules complementaires Firefox.
2. Cliquez sur **Ajouter a Firefox** et confirmez l'invite de permissions.
3. Epinglez l'extension a votre barre d'outils en faisant un clic droit sur l'icone et en selectionnant **Epingler a la barre d'outils**.

{/* TODO: Screenshot of the Firefox Add-ons page for Nostr WoT */}

<Callout type="tip">
Apres l'installation, l'icone de l'extension apparaitra dans la barre d'outils de votre navigateur. Une icone grise signifie que vous n'etes pas encore connecte. Elle devient violette une fois votre identite active.
</Callout>

## Premiere configuration

Lorsque vous ouvrez l'extension pour la premiere fois, l'assistant de configuration vous guide a travers les etapes essentielles. Il vous sera demande de creer une nouvelle identite Nostr ou d'en importer une existante. Aucune inscription aupres d'un tiers n'est necessaire.

{/* TODO: Screenshot of the first-time setup wizard welcome screen */}

## Creer une nouvelle cle Nostr

Si vous etes nouveau sur Nostr, selectionnez **Creer de nouvelles cles**. L'extension genere une cle privee (nsec) et une cle publique (npub) pour vous.

1. Cliquez sur **Creer de nouvelles cles** dans l'assistant de configuration.
2. Sauvegardez votre cle privee. Notez-la ou stockez-la dans un gestionnaire de mots de passe. Si vous perdez cette cle, il n'y a aucun moyen de recuperer votre compte.
3. Definissez un mot de passe local pour chiffrer vos cles dans l'extension. Ce mot de passe n'est jamais envoye nulle part.
4. Remplissez optionnellement les details de votre profil — nom d'affichage, bio et avatar.

{/* TODO: Screenshot of the key creation screen with the backup prompt */}

<Callout type="tip">
Votre cle privee est le mot de passe maitre de votre identite Nostr. Ne la partagez jamais, ne la collez jamais dans un site web et ne la stockez jamais en texte clair. L'extension la chiffre localement pour que les sites ne la voient jamais directement.
</Callout>

## Importer une cle existante

Vous avez deja une identite Nostr ? Selectionnez **Importer une cle existante** et collez votre nsec ou utilisez une extension signataire compatible NIP-07.

1. Cliquez sur **Importer une cle existante**.
2. Collez votre cle privee nsec ou connectez un signataire materiel.
3. Definissez un mot de passe de chiffrement local.
4. L'extension recupere automatiquement votre profil depuis les relais Nostr.

{/* TODO: Screenshot of the import key screen */}

## Verification de votre configuration

Une fois votre identite active, cliquez sur l'icone de l'extension. Vous devriez voir les details de votre profil, votre npub et un graphe de confiance initial qui commence a se construire a partir de votre liste de suivis. L'extension recupere vos contacts depuis les relais et commence a cartographier votre Web of Trust.

Laissez-lui une minute ou deux pour la synchronisation initiale. Apres cela, votre reseau de confiance se met a jour en arriere-plan pendant que vous naviguez.

## Et ensuite ?

Maintenant que l'extension est installee, apprenez a [gerer votre identite](/guides/managing-identity) et comprenez comment le [Web of Trust fonctionne](/guides/understanding-wot).
